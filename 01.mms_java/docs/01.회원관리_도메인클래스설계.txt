## 회원관리시스템
1. 도메인 클래스 분석 및 설계

 
## 회원 property(멤버변수)
1.아이디 문자열 : memberId
2.비밀번호 문자열 : memberPw
3.이름 문자열 : memberName
4.연락처 문자열(기본형식: 010-1234-1234) : mobile
5.이메일 문자열 : email
6.가입일 문자열(기본형식: 년도4자리.월2자리.일2자리) : entryDate
7.등급 문자열(회원종류 : 일반(G), 우수(S), 관리자(A)) : grade
8.마일리지 숫자 : 일반회원 : mileage
9.담당자 문자열 : 우수회원 : manager
